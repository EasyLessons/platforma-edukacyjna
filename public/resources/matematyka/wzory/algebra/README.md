# Wzory Algebraiczne

Pojedyncze wzory algebraiczne do natychmiastowego wstawienia.

## Pliki:

- `wzory-skroconego-mnozenia.webp` - (a+b)², (a-b)², a²-b², ...
- `logarytmy.webp` - Własności logarytmów
- `potegi.webp` - Własności potęg

## Specyfikacja:

- Format: WebP z przezroczystością
- Rozmiar: 800-1000px szerokości
