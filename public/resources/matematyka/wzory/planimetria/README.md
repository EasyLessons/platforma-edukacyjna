# Wzory Planimetria

Pojedyncze wzory i twierdzenia planimetryczne.

## Pliki:

- `twierdzenie-pitagorasa.webp` - a² + b² = c²
- `twierdzenie-talesa.webp` - Proporcje przy prostych równoległych
- `pole-trojkata.webp` - P = 1/2 · a · h
- `wzor-herona.webp` - Wzór Herona na pole trójkąta
- `pole-kola.webp` - P = πr²

## Specyfikacja:

- Format: WebP z przezroczystością
- Rozmiar: 600-800px szerokości
